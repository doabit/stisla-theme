# Stisla bootstrap theme.

## Usage

```shell
yarn add @doabit/stisla-theme
```

```scss
@import '~@doabit/stisla-theme/src/scss/style';
@import '~@doabit/stisla-theme/src/scss/components';
```

## Customization

```scss
$primaryColor: #367ad3f8;
```
